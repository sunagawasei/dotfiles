# カラーシステムドキュメント

このドキュメントは、dotfilesで実際に使用している **Abyssal Teal (深淵の青緑)** テーマのカラーパレット、色選択ルール、および各設定ファイルの色設定一覧を定義します。

**カラー定義の単一ソース**: [`colors/abyssal-teal.toml`](./colors/abyssal-teal.toml)

## 2. 色選択ルール

### 背景系の階調

```
#111E16 → #132018 → #31364C
(最暗)    (メイン)   (パネル)
```

- **`#111E16`**: タブバー背景、最も暗い要素
- **`#132018`**: メイン背景色
- **`#31364C`**: 選択行、パネル背景、ポップアップ、ハイライト

### テキスト系の階調

```
#586269 → #CEF5F2 → #A4E4E0
(補助)    (通常)    (強調)
```

- **`#586269`**: コメント、非アクティブな要素
- **`#CEF5F2`**: 通常のテキスト（メイン）
- **`#A4E4E0`**: 強調テキスト、アイコン

### 意味的色分け

| 意味 | 色 | カラーコード | 使用例 |
|------|-----|-------------|--------|
| 追加/成功 | Teal Green | `#4A8778` | Git追加行、成功メッセージ |
| 変更/警告 | Lavender | `#CED5E9` | Git変更行、警告 |
| 削除/エラー | Purple Muted | `#926894` | Git削除行、エラーメッセージ |
| 情報 | Teal Bright | `#5ABDBC` | 情報メッセージ、パス |
| 文字列 | Teal Mid | `#659D9E` | 文字列リテラル |
| キーワード | Blue Muted | `#54688D` | キーワード、ステートメント |
| 関数 | Teal Bright | `#5ABDBC` | 関数名 |

## 4. 色の一貫性を保つためのガイドライン

### 新しい設定を追加する場合

1. **背景色が必要な場合**:
   - 最暗: `#111E16`
   - メイン: `#132018`
   - パネル/選択: `#31364C`

2. **テキスト色が必要な場合**:
   - 補助: `#586269`
   - メイン: `#CEF5F2`
   - 強調: `#A4E4E0`

3. **アクセントが必要な場合**:
   - プライマリ: `#5ABDBC` (Teal Bright)
   - セカンダリ: `#659D9E` (Teal Mid)
   - エラー/重要: `#926894` (Purple Muted)

### バリデーション

カラー設定を変更した場合は、バリデーションスクリプトを実行して整合性を確認してください:

```bash
cd /Users/s23159/.config/scripts && go run validate-colors.go
```
