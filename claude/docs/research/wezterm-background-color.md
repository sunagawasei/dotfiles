## 3. カラーシステムの設計意図

### Abyssal Teal テーマの特徴

`COLOR-SYSTEM.md` および `colors/abyssal-teal.toml` によると、この背景色は以下の理由で選択されています。

#### 背景階層構造

| 深度 | HEX | 用途 |
|-----|-----|-----|
| 1. Deepest | `#111E16` | 外枠、タブバー背景 |
| **2. Main** | **`#132018`** | **エディタ、ターミナル作業領域** |
| 3. Panel | `#152A2B` | サイドバー、フロート |
| 4. UI Layer | `#31364C` | ポップアップ、非アクティブ |
| 5. Selection | `#64BBBE` | アクティブ選択 |

#### デザインコンセプト

- **「深淵の青緑」**: 深海をイメージした落ち着いた青緑系
- **冷色調**: 長時間作業における目の疲労軽減
- **統一感**: 全ツール（WezTerm、Neovim、Zsh等）で一貫したテーマ

---

## 4. 緑色の色被りの原因

### チャネル分析

```
R: 19 (基準)
G: 32 (+68%)  ← 緑色が突出
B: 24 (+26%)
```

### なぜ緑に見えるか

1. **絶対値が低い**: 非常に暗い色（最大値32）のため、わずかな差異が色調に大きく影響
2. **相対比**: 緑チャネルが他より68%高いため、暗所でも緑味が知覚されやすい
3. **人間の視覚特性**: 人間の目は緑色に最も敏感（錐体細胞の分布特性）

## 5. 純粋なニュートラルダークテーマへの変更案

Geminiによる推奨色（緑色の色被りを除去）:

### オプション1: 同等明度のニュートラルグレー

```lua
-- HEX: #191919
-- RGB: rgb(25, 25, 25)
-- 特徴: 現在の背景と同等の明度、完全な無彩色
```

### オプション2: モダンUIの標準ブラック

```lua
-- HEX: #121212
-- RGB: rgb(18, 18, 18)
-- 特徴: Material Design等で採用される深いニュートラルブラック
```

### 適用方法

`wezterm/wezterm.lua` の以下の箇所を変更:

```lua
-- Before
config.window_background_gradient = {
    colors = { "#132018" },
}
config.colors = {
    background = "#132018",
    -- ...
}

-- After (Option 1: #191919)
config.window_background_gradient = {
    colors = { "#191919" },
}
config.colors = {
    background = "#191919",
    -- ...
}
```

## 6. 推奨事項

### 現在の設定を維持する場合

- 緑色の色被りは **意図された設計** であり、Abyssal Tealテーマのアイデンティティ
- 他のツール（Neovim、Zsh等）との統一感を維持
- 色温度が冷色のため、長時間作業に適している

### ニュートラルテーマに変更する場合

1. **段階的な変更**:
   - まずWezTermのみ `#191919` に変更して数日テスト
   - 問題なければNeovim、Zshなど他のツールも統一

2. **COLOR-SYSTEM.mdの更新**:
   - テーマ名を「Neutral Dark」等に変更
   - `colors/abyssal-teal.toml` を新規ファイルとしてバックアップ

3. **バリデーション**:
   ```bash
   cd scripts && go run validate-colors.go
   ```
